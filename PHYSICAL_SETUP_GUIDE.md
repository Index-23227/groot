# 물리 세팅 가이드 — 약국 조제 보조 시연

> 대회 station의 **로봇과 카메라 위치는 고정**이며, 높이/각도만 조절 가능.
> 소품 배치와 소프트웨어 설정만으로 시나리오를 구성한다.
> 세팅 완료까지 예상 시간: **20~30분**

---

## 0. Station 현황 (고정 사항)

```
⚠️ 변경 불가:
  - 로봇 위치: 테이블 왼쪽 뒤편, 테이블에 볼트 고정
  - 카메라 위치: 테이블 오른쪽 알루미늄 프레임에 마운트
  - 테이블: 흰색 pegboard (구멍 뚫린 작업대), 약 80×60cm

✅ 변경 가능:
  - 카메라 높이 (프레임 위아래 슬라이드)
  - 카메라 각도 (틸트 조절)
  - 소품 위치 (테이블 위 자유 배치)
  - 그리퍼 end-effector (이미 2핑거 그리퍼 장착됨)
```

## 1. 테이블 배치도

### 위에서 본 배치 (Top View)

```
   알루미늄 프레임 (카메라 마운트)
   │                                               │
   │  ┌──────────────────────────────────────────┐  │
   │  │                                          │  │
   │  │   [A] 약품 대기 구역     [B] 조제함       │  │
   │  │                                          │  │
   │  │   ┌───┐  ┌───┐  ┌───┐   ┌──┐┌──┐┌──┐   │  │
   │  │   │ 파 │  │ 빨 │  │ 하 │   │1 ││2 ││3 │   │  📷
   │  │   │ 랑 │  │ 강 │  │ 양 │   └──┘└──┘└──┘   │  카메라
   │  │   └───┘  └───┘  └───┘                    │  (프레임 상단)
   │  │                                          │  │
   │  │   ┌───┐                                  │  │
   │  │   │ 노 │                                  │  │
   │  │   │ 랑 │                                  │  │
   │  │   └───┘                                  │  │
   │  │                                          │  │
   │  │  🦾 로봇 베이스                            │  │
   │  │  (테이블 왼쪽 뒤,                          │  │
   │  │   볼트 고정)                               │  │
   │  └──────────────────────────────────────────┘  │
   │                                               │
   └───────── 흰색 pegboard 테이블 ─────────────────┘
```

### 옆에서 본 배치 (Side View)

```
        알루미늄 프레임
        │
        │  📷 카메라 (높이/각도 조절 가능)
        │ ╱   ↕ 슬라이드 가능
        │╱    45~60° 하향 권장
        │
   ┌────┴──────────────────────────┐
   │  🦾    약품     조제함         │  ← pegboard 테이블
   │ (고정)                        │
   └───────────────────────────────┘
```

### 배치 핵심 원칙

```
1. 약품은 로봇 팔 바로 앞 (베이스에서 25~40cm)
   → reach 안에 있어야 하고, 카메라에도 보여야 함

2. 조제함은 약품 오른쪽 (카메라 쪽)
   → 로봇이 왼쪽→오른쪽으로 이동하는 동선
   → 카메라에서 조제함 번호가 잘 보이는 위치

3. 약품과 조제함 사이에 장애물 없이 빈 공간 확보
   → 로봇 팔 이동 경로에 충돌 요소 없어야 함

4. 테이블 pegboard 구멍을 기준점으로 활용
   → 소품 위치 재현성 확보 (구멍 번호로 기록)
```

## 2. 소품 준비 체크리스트

### 약품 (대용품)

| 약품 | 대용품 제안 | 크기 | 요구사항 |
|------|-----------|------|---------|
| 파란 약병 | 파란 물병, 파란 마커 붙인 공병 | 높이 10~12cm, 지름 4~5cm | 2핑거 그리퍼로 잡을 수 있는 크기 |
| 빨간 약병 | 빨간 물병, 사진 속 핑크 물병도 가능 | 동일 | 파란색과 명확히 구분 |
| 하얀 약병 | 하얀 공병, 종이컵 (사진 속 종이컵 활용 가능) | 동일 | 흰색 테이블 위라 색 테이프로 구분 표시 필요 |
| 노란 약상자 | 노란 포스트잇 상자, 약상자 | 5×8×3cm | 병과 다른 형태 |

```
⚠️ 현장 주의:
- 테이블이 흰색 → 하얀 물체는 구분 어려움
  → 하얀 약병 대신 초록/주황 등 대비 높은 색 권장
- 사진에 보이는 종이컵/가위 등은 시연 시 치워야 함
- 그리퍼가 2핑거 → 지름 4~5cm 물체가 적합
  → 현장의 그리퍼 열림 폭을 먼저 측정!
```

### 조제함 (대용품)

```
권장: pegboard 구멍에 맞춘 마스킹 테이프 구역

  ┌──────────────────────────────────┐
  │   ┌─────┐  ┌─────┐  ┌─────┐    │
  │   │  1  │  │  2  │  │  3  │    │  ← 마스킹 테이프로 구역 표시
  │   │     │  │     │  │     │    │     각 10×10cm
  │   └─────┘  └─────┘  └─────┘    │     번호는 A4에 매직으로 크게
  └──────────────────────────────────┘

pegboard 장점:
  - 구멍 위치로 정확한 좌표 재현 가능
  - "구멍 3개 간격" 같은 기준으로 팀원 공유
  - 테이프가 잘 붙음
```

### 기타 필요 물품

```
□ 마스킹 테이프 (구역 표시, pegboard에 잘 붙음)
□ 유성 매직 + A4 용지 (1번, 2번, 3번 라벨)
□ 색 테이프 (하얀 물체 구분용, 물체에 감기)
□ 마이크 (STT용, 노트북 내장도 가능)
□ (선택) 블루투스 스피커 (TTS 출력용)

이미 있는 것 (station 제공):
  ✅ 카메라 (알루미늄 프레임에 마운트됨)
  ✅ 로봇 (테이블에 고정)
  ✅ 그리퍼 (2핑거, 장착됨)
  ✅ 테이블 (pegboard)
```

## 3. 카메라 세팅

### 고정 위치에서의 조절

```
카메라가 알루미늄 프레임 오른쪽에 고정되어 있으므로:

조절 가능한 것:
  ✅ 높이: 프레임 위아래로 슬라이드
     → 권장: 테이블 위 40~60cm (물체를 45~60° 각도로 내려다봄)

  ✅ 각도 (틸트): 카메라 마운트의 틸트 조절
     → 권장: 45~60° 하향
     → 너무 수직이면 물체 높이 구분 어려움
     → 너무 수평이면 테이블 뒤쪽이 안 보임

조절 불가:
  ✗ 좌우 위치 (프레임에 고정)
  ✗ 전후 위치 (프레임에 고정)

핵심: 카메라가 오른쪽에 있으므로
  → 약품을 로봇 바로 앞 (카메라에서 보면 왼쪽)에 배치
  → 조제함을 카메라 쪽 (오른쪽)에 배치
  → 로봇 팔이 물체를 가리는 occlusion 최소화
```

### 카메라 높이/각도 결정 절차

```
1. 카메라를 프레임 중간 높이에 놓고 시작
2. 프리뷰 실행 (아래 명령)
3. 약품 4개 + 조제함 3개를 배치한 상태에서 확인:
   □ 모든 약품이 프레임 안에 보이는가?
   □ 모든 조제함 슬롯이 보이는가?
   □ 로봇 그리퍼가 약품 위에 있을 때 보이는가?
   □ 약품 색상이 구분되는가?
4. 안 보이면 높이를 올리거나 각도를 더 수직으로
5. 최적 위치 찾으면 프레임 클램프 단단히 조임
```

### 화각 확인

```bash
# 카메라 프리뷰
python -c "
import cv2
cap = cv2.VideoCapture(0)
cap.set(cv2.CAP_PROP_FRAME_WIDTH, 640)
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 480)
while True:
    ret, frame = cap.read()
    if not ret: break
    cv2.imshow('Camera Check', frame)
    if cv2.waitKey(1) == ord('q'): break
cap.release()
"
```

### 카메라 체크리스트

```
□ 약품 대기 구역 전체가 프레임 안에 보이는가?
□ 조제함 전체가 프레임 안에 보이는가?
□ 로봇 그리퍼가 프레임 안에 보이는가?
□ 각 약품의 색상이 구분되는가?
□ 조제함 번호 라벨이 읽히는가?
□ 카메라 클램프가 단단히 조여 있는가? (흔들림 없이)
□ 카메라 케이블이 로봇 동선에 걸리지 않는가?
□ 프레임 자체가 로봇 동선을 방해하지 않는가?
```

### 조명 주의사항

```
⚠️ 현장 형광등 환경:
- 흰색 테이블 + 형광등 → 반사광 주의
  → 카메라 각도로 반사 회피
- 파란색과 하얀색이 비슷하게 보일 수 있음
  → 대비 높은 색상 소품 선택
- 로봇 팔(흰색)과 흰색 물체 구분
  → 흰색 약병은 피하거나, 색 테이프로 감싸기
```

## 4. 로봇 세팅

### 고정 위치에서의 작업 범위

```
로봇이 테이블 왼쪽 뒤에 고정이므로:

E0509 제원:
  Reach: 900mm
  Payload: 5kg

사용 가능 영역 (로봇 베이스 기준):
  ┌──────────────────────────┐
  │  🦾                      │
  │  ╱ ╲                     │
  │ ╱   ╲  ← reach 900mm    │
  │╱ 사용 ╲  반경            │
  │  가능   ╲                │
  │  영역    ╲               │
  └──────────────────────────┘

  → 로봇 앞쪽 20~50cm 범위에 소품 배치
  → 너무 오른쪽(카메라 프레임 쪽)은 충돌 위험
  → 너무 뒤(로봇 뒤)는 접근 어려움

작업 범위 테스트:
  1. teach pendant로 약품 각 위치까지 수동 이동
  2. 조제함 각 슬롯까지 수동 이동
  3. 중간 경로에 프레임/케이블 등 장애물 없는지 확인
  4. 도달 불가하면 소품 위치를 로봇 쪽으로 조정
```

### 그리퍼 확인

```
사진에서 확인: 검정색 2핑거 그리퍼 장착됨

현장에서 확인할 것:
  1. 그리퍼 열림 폭 측정 (자로 재기)
     → 약품 지름이 이 폭보다 작아야 함
  2. 제어 방식 확인 (I/O / ROS2 topic / DRCF)
  3. 파지 힘 확인 → 약병이 깨지지 않을 정도
  4. 그리퍼와 물체 사이 마찰 테스트
     → 미끄러우면 고무 패드 부착

  → configs/doosan_e0509_config.py 업데이트:
    GRIPPER_TYPE = "2finger"  # 실제 모델명으로
```

### Joint Limits 실측

```bash
# 현재 config 확인
python -c "
from configs.doosan_e0509_config import *
print_config()
"
# teach pendant로 실측한 값과 비교 → 차이 있으면 config 수정
```

## 5. 데모 수집 가이드

### 수집 전략

```
총 에피소드: 50~70개 (약 1.5~2시간)

물체별 배분:
  파란 약병 → 1번: 10개
  파란 약병 → 2번: 5개
  파란 약병 → 3번: 5개
  빨간 약병 → 2번: 10개
  빨간 약병 → 1번: 5개
  하얀 약병 → 3번: 10개
  노란 약상자 → 1번: 5개
  노란 약상자 → 2번: 5개
  ──────────────────────────
  합계: ~55개 + 여유 15개 (실패/불량 대비)

⚠️ 매 에피소드마다 약품 위치를 ±3~5cm 변동시키기
   (pegboard 구멍 1~2개 차이)
   → 모델의 위치 일반화 능력 향상
```

### 수집 절차

```
1. 약품을 대기 구역의 약간 다른 위치에 배치 (매번 ±3~5cm)
2. 터미널에서 instruction 확인
3. [Enter] → hand-guiding 버튼 누르고 동작 수행:
   정지 → 약품으로 이동 → 그리퍼 닫기 → 들기 →
   슬롯으로 이동 → 놓기 → home 복귀
4. [Ctrl+C] → 저장
5. 약품 원래 위치로 리셋
6. 반복

수집 명령:
  python utils/doosan_recorder.py \
    --task "Pick up the blue medicine bottle and place it in dispensing slot 1" \
    --num-episodes 10

⚠️ hand-guiding 시 주의:
  - 카메라 프레임(알루미늄)에 팔이 부딪히지 않도록
  - 케이블에 걸리지 않도록
  - 일정한 속도로 부드럽게 움직이기
```

### 수집 품질 체크

```
좋은 데모:
  ✓ 일정한 속도 (급가속/급감속 없음)
  ✓ 직선에 가까운 경로 (불필요한 우회 없음)
  ✓ 물체 위에서 정확히 그리퍼 닫기
  ✓ 슬롯 위에서 정확히 그리퍼 열기
  ✓ 5~10초 내 완료
  ✓ 카메라 프레임/케이블 충돌 없음

나쁜 데모 (제거):
  ✗ 10프레임 미만 (너무 짧음)
  ✗ 중간에 멈칫거림/흔들림
  ✗ 물체를 떨어뜨림
  ✗ 잘못된 슬롯에 넣음
  ✗ 프레임에 부딪힘

데모 검증:
  python utils/doosan_recorder.py --verify --data-dir ./data/raw
```

## 6. 현장 세팅 순서 (타임라인)

```
[0:00 ~ 0:05] Station 상태 확인
              → 로봇 전원 ON, 연결 확인
              → 카메라 연결 확인 (USB)
              → 그리퍼 동작 확인
              → 테이블 위 불필요한 물건 치우기

[0:05 ~ 0:10] 작업 범위 테스트
              → teach pendant로 테이블 각 구역까지 이동
              → 도달 가능 영역 파악
              → 프레임/케이블 충돌 위험 구간 파악

[0:10 ~ 0:15] 소품 배치
              → 약품 대기 구역 결정 (로봇 앞 25~40cm)
              → 조제함 위치 결정 (약품 오른쪽)
              → 마스킹 테이프로 위치 표시
              → pegboard 구멍 번호 기록 (재현성)

[0:15 ~ 0:20] 카메라 높이/각도 조정
              → 프리뷰로 화각 확인
              → 모든 소품 + 그리퍼가 보이는지 확인
              → 클램프 단단히 조이기

[0:20 ~ 0:25] config 업데이트
              → CONTROLLER_IP, GRIPPER_TYPE, CAMERA_INDEX
              → joint limits 실측값 반영
              → 그리퍼 열림 폭 → 물체 크기 최종 결정

[0:25 ~     ] 데모 수집 시작
```

## 7. 트러블슈팅

### 물체를 못 잡음

```
원인 1: 그리퍼 열림 폭 < 물체 지름
  → 더 작은 물체로 교체 (그리퍼 폭 -1cm 이하)

원인 2: 물체가 미끄러움
  → 고무 밴드 감기, 마스킹 테이프 붙이기

원인 3: VLA가 물체 위치를 잘못 인식
  → 카메라 각도 미세 조정
  → 데모 추가 수집 (해당 위치)
```

### 카메라에서 물체가 안 보임

```
원인 1: 물체가 카메라 화각 밖에 있음
  → 카메라 높이를 올려서 화각 넓히기
  → 또는 물체를 카메라 쪽(오른쪽)으로 이동

원인 2: 로봇 팔이 물체를 가림
  → 물체를 로봇 앞쪽(카메라에서 멀어지는 방향)으로 이동
  → 카메라 각도를 더 수직으로 조정

원인 3: 프레임 기둥이 시야를 가림
  → 소품 배치를 프레임 안쪽으로 조정
```

### 색상 구분 안 됨

```
원인: 형광등 + 흰색 테이블에서 색상이 비슷하게 보임
  → 더 대비 높은 소품으로 교체
  → 물체에 색 테이프 감기
  → 카메라 화이트밸런스 수동 조정

⚠️ 특히 주의: 흰색 테이블 위의 흰색 물체
  → 하얀 약병은 초록/주황 등으로 교체 권장
```

### 로봇 동선에서 충돌

```
원인 1: 알루미늄 프레임에 팔이 부딪힘
  → 소품을 프레임에서 먼 쪽(왼쪽)으로 이동
  → Home 자세를 프레임 반대 방향으로 설정

원인 2: 카메라 케이블에 걸림
  → 케이블을 프레임에 타이랩으로 고정
  → 케이블 여유분을 프레임 위로 올려놓기

원인 3: 약품과 조제함 사이 이동 시 테이블 장비에 부딪힘
  → 이동 경로 위의 모든 장비 치우기
  → 필요하면 약품/조제함 간격 좁히기
```

### pegboard 위에서 물체가 불안정

```
원인: pegboard 구멍에 물체 바닥이 걸림
  → 물체 아래에 얇은 종이/카드보드 깔기
  → 또는 구멍 없는 영역에 배치
  → 마스킹 테이프로 구멍 막기
```
